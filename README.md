# Coffee-Shop-sales
# ☕ Coffee Shop Sales Analysis (Excel Project)

## 📌 Project Overview
This project focuses on analyzing coffee shop retail sales data to uncover meaningful insights that can help improve business performance and decision-making.

---

## 🎯 Objective
The main objective of this project is to:
- Analyze sales trends
- Identify peak business hours
- Understand customer purchasing behavior
- Highlight top-performing products and categories

---

## 📊 Key Business Questions
- How do sales vary by **day of the week** and **hour of the day**?
- What are the **peak sales hours**?
- What is the **total monthly revenue**?
- How do sales differ across **store locations**?
- What is the **average order value per customer**?
- Which products are **best-selling** (by quantity and revenue)?
- How do sales vary by **product category and type**?

---

## 🛠 Tools & Techniques Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Data Cleaning
  - Conditional Formatting
  - Formulas (SUM, IF, INDEX-MATCH)

---

## 📈 Key Insights
- Identified peak hours where sales are highest
- Found top-selling products contributing most to revenue
- Observed variation in performance across locations
- Discovered customer purchasing patterns and average spending

---

## 📁 Dataset
- Coffee shop sales dataset (retail transactions)
- Includes: date, time, product, category, price, location

---

## 🚀 How to Use
1. Download the Excel file
2. Open in Excel (2016 or later)
3. Explore Pivot Tables and Dashboard
4. Interact with filters/slicers to view insights

---

## 📌 Conclusion
This project demonstrates how Excel can be used as a powerful tool for data analysis, even for beginners. It highlights the importance of transforming raw data into actionable insights.
